* Ender-3 [g-code](./ManetaNevera/CE3_ManetaNevera_xtrFast_Cubicsub.gcode). Slicer: Cura 4.1.0.
	- Print Settings : Extra Fast - 0.3mm.
	- Infill pattern : Cubic Subdivision.
	- Infill Density : 20%.
	- Plate adhesion : None.
	- Printing Time : 1:30h aprox.
